# Tensorrt-yolov3-win10
onnx-tensorrt for yolov3  
thanks for lewes6369/TensorRT-Yolov3 and official sample by tensorrt  
## environment  
windows10  
cuda9.0  
cudnn7.3.1

vs2017

| ----------- | :--------: | :--------: | :------------: |
| yolov3      | 1          | 416x416    | 25ms           |
| yolov3-tiny | 1          | 608x608    | 7ms            |
